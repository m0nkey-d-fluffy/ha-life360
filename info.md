# <img src="https://brands.home-assistant.io/life360/icon.png" alt="Life360" width="50" height="50"/> Life360

A [Home Assistant](https://www.home-assistant.io/) integration for Life360.

## Features

- **Member Tracking** - Creates `device_tracker` entities for all Life360 Circle members
- **Tile Support** - Track Tile Bluetooth devices linked to your Life360 account
- **Pet GPS Support** - Track Jiobit pet/child GPS trackers
- **Driving Detection** - Know when family members are driving
- **Location Updates** - Request on-demand location updates
